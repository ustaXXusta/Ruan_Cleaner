# MacCleanerPro

A safe, fast, modern macOS cleaner that finds and removes junk, caches, logs, duplicates, old downloads, unused languages, etc., while NEVER breaking the system or user data.

## Features
- **Scan & Analysis**: Comprehensive scanning of system caches, logs, temporary files, and more.
- **Safety First**: Dry-run mode, SIP protection check, and dependency checker.
- **Cleanup Categories**: Granular control over what to clean.
- **Modern UI**: Simple, beautiful, and dark-mode ready interface.
- **Privacy**: Zero telemetry, all processing local.

## Architecture
- **Language**: Swift
- **UI**: SwiftUI
- **Architecture**: MVVM
