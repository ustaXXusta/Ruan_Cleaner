# Installing Ruan Cleaner

## Prerequisites
- macOS 12.0 or later
- Swift 5.5 or later (installed via Xcode or Swift toolchain)

## Quick Start
1. Open Terminal.
2. Navigate to this folder.
3. Run the setup script:
   ```bash
   chmod +x setup.sh
   ./setup.sh
   ```

## Manual Build
If you prefer to build manually:
```bash
swift build -c release
```

## Running the App
```bash
swift run -c release
```
